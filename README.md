# ChatterBot


## Installation

This package can be installed from [PyPi](https://pypi.python.org/pypi/ChatterBot) by running:

```
pip install chatterbot -> for windows from version 3.6
```


```
pip3 install chatterbot -> for linux from version 3.x

```
